## 最重要的资源规则

- 用户提供的 UI 切图都按 `750x1334` 设计稿尺寸制作。
- 前端基准是 `375x667 CSS px`。
- 所有切图默认只做 `0.5` 倍缩放，不要未经用户确认改尺寸。
- 新资源会按页放进 `D:\home for cc\小火箭设计资产\pX` 或 `P X/PX` 目录。
- 同步到项目时放入 `public/assets/pX`。
- 小步 UI 调整时，默认只跑 `npm.cmd run build`，由用户自己刷新预览；只有复杂动画/定位需要确认时再跑浏览器自动预览。

- 已接入第一版。
- P4 发射后镜头上移进入 P5，小火箭缩小上升。
- 资源目录：`public/assets/p5`。
- 背景装饰包括蓝色星球、完整黑色星球、红色星星、黄色星星。
- 当前黑色星球使用：`完整黑色星球.svg`，尺寸按 `668.16 / 2 = 334.08px`。
- P5 只显示分类为 `我能行动` 和 `我能影响` 的碎片。
- 分类为 `我管不了` 的碎片需要留到 P6 出场。
- 蓝色建议云：
  - 点击展开/收起详细建议。
  - 左滑触发当前云组离场。
- 红色碎片云：
  - 右滑触发当前云组离场。
- 离场时蓝云向左飞、红云向右飞。
- 云组划开后，红/黄装饰星星轻微弹动，火箭上冲，尾焰拉长再恢复。
